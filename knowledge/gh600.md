# GitHub Copilot Certification (GH-600)

## Qué es
Certificación oficial de GitHub Copilot — track de experto. Es el examen al que apunta el bootcamp Harness Engineering como complemento.

## Examen
- Formato: online proctored vía OnVUE
- Duración: 120 minutos
- Idioma: solo inglés
- Costo: USD 99 (Jack pagó USD 54 con código beta GH600Flanders)
- Resultados beta: ~8 semanas post cierre de período beta

## Estructura — 6 dominios (actualizado mayo 2026)
- **D1**: Agent architecture + SDLC — 15-20%
- **D2**: Tools + environment — 20-25%
- **D3**: Memory, state, execution — 10-15%
- **D4**: Eval + error analysis — 15-20%
- **D5**: Multi-agent orchestration — 15-20% (nuevo)
- **D6**: Guardrails + responsibility — 10-15% (nuevo)

## Relación con el bootcamp
El bootcamp NO da la certificación. La certificación se obtiene rindiendo el examen en OnVUE.

El bootcamp **prepara** para los 6 dominios con contenido aplicado en español. Una semana del bootcamp = un dominio del examen.
